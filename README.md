## 🙌 서비스 소개 

그룹이룸은 팀 기반 게임의 팀원을 모집하기 위한 플랫폼입니다.

팀원을 모집하기 위해 직접 글을 작성하거나, 다른 팀에 가입하기 위해

모집글에 첨부된 연락처(디스코드, 오픈채팅)로 작성자와 연락할 수 있습니다.
